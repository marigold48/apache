Kernel

Definición

El Kernel de RETO, reúne las funcionalidades comunes que necesita cualquier aplicación

Es un conjunto de librerias javascript, basado en el standard ES6

Se utiliza el mecanismo de módulos de ES6, para usar las funciones y clases definidas, desde los módulos .js de la aplicación

Ajax

Contiene las funciones que permiten las llamadas asíncronas al server

  • Grabar una topología en MongoDB
  • Obtener una topología mediante su _id
  • Actualizar una topología existente
  • Borrar una topología mediante su _id
  • Obtener una lista de objetos meta de las topologías, por su {iam}
  • Obtener una lista de objetos meta de las topologías, por su {iam} y organización {org}
  • Ejecutar scripts tipo Bash, Python, etc en el server

Clases auxiliares

Contiene las clases que no son de Topología

  • Clases para el módulo de Multi Lenguaje (ML)
  • Arbol de cálculos
  • Arbol BTree+
  • Etc

Cultivos

Planificación de cultivos de variedades sobre los diferentes bancales

  • Creación escenarios (matriz bancales/variedades), en un tiempo definido
  • Asignación de variedades a bancales
  • Edición del Grafo de tareas previstas
  • Temporadas
  • Esquema Gantt de tareas
  • Agenda de tareas (calendario)

Riego

Infraestructuras de riego de los cultivos en los diferentes bancales

  • Captación del agua (pozos, lluvia, etc)
  • Acumulación (depósitos, embalses, etc)
  • Transporte (canales, tuberías, etc)
  • Red de distribución
  • Métodos (aspersión, goteo, etc)

Evaluaciones

Una vez planificado una temporada, se puede evaluar con diferentes ópticas:

  • Evaluación económica prevista de la temporada
  • Evaluación de incompatibilidades entre productos
  • Evaluación de rotaciones
  • Planificación de riegos
    • Necesidades hídricas de cultivos
    • Previsión del calendario de riegos
    • Ajustes del calendario de riegos por climatología
