Topología : Nodos

Resumen

El Nodo es la Clase con la que crear las topologías (Conjt, Lista, Arbol, Grafo y Malla).

Básicamente un Nodo se compone de un identificador id0, y de un rótulo o tag.

Además, por conveniencia, consta de una propiedad iam, con el nombre de la clase, y de rol, que indica qué tratamiento ha de tener

Las propiedades, por tanto son:

  • id0 : identificador para asegurar unicidad dentro de la topología
  • tag : rótulo que aparece en su representación en pantalla
  • iam : nombre de la clase
  • rol : por defecto 'NODO'. En Mallas puede ser ROW ó COL
  • lng : objeto para propósito de Multi Lenguaje

Creación

Se puede crear un nuevo Nodo 'Nodo 1' invocando: 

var nodo1 = new rNodo('Nodo 1');

La clase rNodo es (abrev.): 

class rNodo {
  constructor(tag){
    this.id0 = vgk.idsNodo.getId(); // (*)
    this.tag = tag || ('Nuevo');
    this.iam = 'rNodo';
    this.rol = 'NODO';
    this.lng ={};
  }
}

(*) Los id0 se generan aleatoriament, a partir de una clase: 

class rIds {
  constructor(prec,serie){
    this.prec = prec || 6;
    this.serie = serie || 1;
  }

  getId(){
    var base = 1;
    for (var i=0;i<this.prec;i++) base *= 10; // 10 ^ prec
    return Math.floor(Math.random()*base + this.serie*base);
  }
}

vgk = {
  idsNodo : new rIds(6,1), // generador de id0's para nodos;
}
