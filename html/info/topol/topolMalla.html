Topología Malla

Resumen

La topología de Malla, se basa en Grafo, o sea, emplea para modelizar un conjunto de entidades, relacionadas por arcos entre ellas.

La diferencia es que el Grafo permite arcos entre cualquier par de nodos

En una Malla, los arcos (Nudos) van de un conjunto de nodos (rows) a otro (cols)

Las funciones de dicha topología serán, entre otras:

  • Creación
  • Añadir Nodos ROW|COL.
  • Actualizar Nodo.
  • Añadir Nudo.
  • Actualizar Nudo.
  • Borrar Nodo ROW|COL (y los Nudos involucrados).
  • Borrar Nudo.

Creación

Se puede crear una nueva Malla XYZ invocando: 

var malla1 = new rMalla('XYZ',[]);

Posteriormente se crean los nodos ROW|COL y los nudos: 

var row1 = new rNodo('Row 1'); row1.rol = 'ROW';
malla1.addNodo(row1);
var row2 = new rNodo('Row 2'); row2.rol='ROW';
malla1.addNodo(row2);
var row3 = new rNodo('Row 3'); row2.rol='ROW';
malla1.addNodo(row3);

var col1 = new rNodo('Col 1'); col1.rol='COL';
malla1.addNodo(col1);
var col2 = new rNodo('Col 2'); col2.rol='COL';
malla1.addNodo(col2);
var col3 = new rNodo('Col 3'); col3.rol='COL';
malla1.addNodo(col3);

var nudo12 = new rNudo('N12',row1,col2,'valor 1.2');
malla1.addNudo(nudo12);
var nudo23 = new rNudo('N23',row2,col3,'valor 2.3');
malla1.addNudo(nudo23);

Quedará una Malla, con 3 nodos ROW, 3 nodos COL y 2 nudos:

Col 1Col 2Col 3
Row 1Valor 1.2
Row 2Valor 2.3
Row 3
