Topología Grafo

Resumen

La topología de Grafo, se emplea para modelizar un conjunto de entidades, relacionadas por arcos entre ellas.

Las funciones de dicha topología serán, entre otras:

  • Creación
  • Añadir Nodo.
  • Actualizar Nodo.
  • Añadir Arco.
  • Actualizar Arco.
  • Borrar Nodo (y los Arcos involucrados).
  • Borrar Arco.
  • Recorrer Grafo (varios algoritmos)

Creación

Se puede crear un nuevo Grafo XYZ invocando: 

var grafo1 = new rGrafo('XYZ',[]);

Posteriormente se crean los nodos y los arcos: 

var nodo1 = new rNodo('Nodo 1');
grafo1.addNodo(nodo1);
var nodo2 = new rNodo('Nodo 2');
grafo1.addNodo(nodo2);
var nodo3 = new rNodo('Nodo 3');
grafo1.addNodo(nodo3);
var arco12 = new rArco('A12',nodo1,nodo2);
grafo1.addArco(arco12);
var arco23 = new rArco('A23',nodo2,nodo3);
grafo1.addArco(arco23);

Quedará un Grafo,con 3 nodos y 2 arcos: nodo1 ---> nodo2 ---> nodo3
