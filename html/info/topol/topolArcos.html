Topología : Arcos

Resumen

El Arco es la Clase con la que crear las relaciones de topología en Grafos.

Básicamente un Arco es como un Nodo, y se compone de dos identificadores id0 y id1, con el rótulo o tag.

Además, por conveniencia, consta de una propiedad iam, con el nombre de la clase, y de rol, que indica qué tratamiento ha de tener

Las propiedades, por tanto son:

  • id0 : identificador que apunta al nodo Origen
  • id1 : identificador que apunta al nodo Target
  • tag : rótulo que aparece en su representación en pantalla
  • iam : nombre de la clase
  • rol : por defecto 'ARCO'.

Creación

Se puede crear un nuevo Arco 'Arco 1-2' invocando: 

var nodo1 = new rNodo('Nodo 1');
var nodo2 = new rNodo('Nodo 2');
var arco1 = new rArco('Arco 1-2',nodo1,nodo2,0);

La clase rArco es (abrev.): 

class rArco {
  constructor(tag,nodo0,nodo1,n){
    this.tag = tag || 'x';
    this.iam = 'rArco';
    this.rol = 'ARCO';
    this.num = n || 0; // para permitir varios arcos entre dos nodos dados
    if (nodo0 && nodo1){
      this.id0 = ((nodo0.id0+123) ^ nodo1.id0) + this.num; // (*)
      this.id1 = nodo1.id0;
    }
    else {
      this.id0 = 0;
      this.id1 = 0;
    }
  }

(*) El id0 del Arco se recalcula, para asegurar su unicidad, por si hay varios arcos con el mismo nodo origen.

Se efectúa un XOR con el id1, sumado un número arbitrario (123, para diferenciar A --> B de B--> A)

Para obtener el id0 real del nodo origen, hay un método:

 getId0Real(){
    return ((this.id0 ^ this.id1)-123)-this.num;
  }
