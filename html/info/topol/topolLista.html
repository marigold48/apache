Topología Lista

Resumen

La topología de Lista, se emplea para modelizar un conjunto de entidades, relacionadas por un orden entre ellas.

Las funciones de dicha topología serán, entre otras:

  • Creación
  • Añadir Nodo.
  • Actualizar Nodo.
  • Borrar Nodo.
  • Ordenar Lista

Creación

Se puede crear una nueva Lista XYZ invocando: 

var lista1 = new rlista('XYZ',[]);

Posteriormente se crean los nodos: 

var nodo1 = new rNodo('Nodo 1');
lista1.addNodo(nodo1);
var nodo2 = new rNodo('Nodo 2');
lista1.addNodo(nodo2);
var nodo3 = new rNodo('Nodo 3');
lista1.addNodo(nodo3);

O también: 

var nodo1 = new rNodo('Nodo 1');
var nodo2 = new rNodo('Nodo 2');
var nodo3 = new rNodo('Nodo 3');
var lista1 = new rlista('XYZ',[nodo1,nodo2,nodo3]);
